    [sarahebourne] @virtualalex The feds have recently updated Section 508, the regulations for federal agencies (and others receiving federal funding) and it now includes by reference WCAG2 AA for technical requirements (instead of making them up themselves.) So WCAG2 AA compliant is Section 508 compliant. The effort to set regulations for web accessibility under the ADA has been sidelined by the current administration, so there is currently no definition of web accessibility under the ADA. As @ugi mentioned, settlements for web-related ADA cases have uniformly required WCAG2 AA. While there are no guarantees (and I am not a lawyer) using WCAG2 AA should keep you free of any credible lawsuits.
